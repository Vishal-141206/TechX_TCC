```

2. Group Data ‚Üí By merchant, category, date
3. Detect Patterns ‚Üí Recurring subscriptions
4. Analyze Trends ‚Üí Increasing/Stable/Decreasing
5. Forecast ‚Üí Next month income & expenses
6. Generate Insights ‚Üí Personalized advice
```

### Prediction Formula

```kotlin
// Income
predictedIncome = average(recentCredits)

// Expenses
predictedExpenses = (historicalSpending √ó 0.6) + (recurringPatterns √ó 0.4)

// Balance
```
